# Real time Weather Dashboard

---


## Tech Stack
- **Frontend:** React.js
- **Backend:** Node.js, Express.js
- **API:** OpenWeatherMap

## Setup server
```
cd server
npm install
```

## Setup client
```
cd client
npm install
```

# Run the App

## Start Backend
```
cd server
node server.js
```

## Start Frontend
```  
cd client
npm start
```
